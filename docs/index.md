A Python web platform for running predictive analytics and machine learning experiments.

## Highlights

- Multipage Dash application built with Dash Mantine Components and AG Grid
- Modular layout and component architecture
- Built-in managers for state, caching, and persistence

## Quick start

```bash
uv sync
uv run python -m aiml_studio.app
```

Visit `http://localhost:8050` to explore the experience.
